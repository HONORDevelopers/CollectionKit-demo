This folder contains libraries copied from the "CollectionKitTest" project.
It is managed by the Honor-codechecks-idea IDE plugin.
Do not modify this folder while the IDE is running.
When the IDE is stopped, you may delete this folder at any time. It will be recreated as needed.
In order to prevent the Honor-codechecks-idea IDE plugin from creating this folder,
uncheck the "Copy libraries from project directory" option in the Honor-codechecks-idea settings dialog.
